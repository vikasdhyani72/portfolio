Description:

4. UI/UX Design: Focused on user interface design to create a visually appealing and user-friendly navigation system.
5. Version Control (Git): Managed the project using Git for version control, allowing for efficient collaboration and change tracking.

Skills Used:
1.HTML: Structuring the webpage and defining elements for interaction.

Project 3: Key Press Program

Description:
The Key Press Program is an interactive web application that captures and displays the key pressed by the user inside a text box. When a user types any key, the program dynamically updates the displayed key and its corresponding key code, providing instant feedback. This program is designed to enhance understanding of keyboard events in web development and improve user interaction.
Skills Used:
1.HTML: Structure of the webpage and input elements.
2.CSS: Styling the layout and enhancing user experience with a responsive design.
3.JavaScript: Event handling to capture key presses and update the DOM dynamically.
